# 🛡️ Política de Seguridad (Security Policy)

Esta EdTech se compromete con la protección de datos y el cumplimiento de normativas educativas internacionales.

## 🛑 Reporte de Vulnerabilidades

**NO abras un Issue público para reportar un fallo de seguridad.**

Para reportar vulnerabilidades de forma confidencial:
1. Envía un correo electrónico a `dev@apolo.education` (o tu canal de reporte designado).
2. Incluye una descripción detallada, pasos para reproducir y el impacto potencial.
3. Responderemos en un plazo máximo de 48 horas con el plan de mitigación.

Este documento es accesible desde la sección de seguridad de todos los repositorios de esta EdTech.
