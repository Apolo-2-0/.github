# 📝 Plantillas de Incidencias (Issue Templates)

Utiliza este espacio para centralizar los formularios de captura de error y mejora para toda la organización.

## 📁 Contenido Recomendado

- **`bug_report.yml`**: Reportes de errores técnicos con contexto (OS, versión, logs).
- **`feature_request.yml`**: Estructurar los nuevos casos de uso.

Cualquier cambio aquí se propagará automáticamente a los repositorios de **`backend`**, **`frontend`** y **`docs`**.
